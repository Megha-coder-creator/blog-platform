# 📝 BlogSphere

A modern full-stack blogging platform built using the MERN stack. BlogSphere allows users to register, log in securely, create blogs, edit or delete their own posts, and explore articles shared by the community.

---

## 🚀 Features

- 🔐 User Authentication (JWT)
- 👤 User Registration & Login
- ✍️ Create Blog Posts
- 📝 Edit Your Posts
- 🗑️ Delete Your Posts
- 📖 View All Blogs
- 📄 Read Full Blog Details
- 📱 Responsive UI
- 🎨 Clean and Modern Interface

---

## 🛠️ Tech Stack

### Frontend
- React.js
- React Router DOM
- Bootstrap
- Axios

### Backend
- Node.js
- Express.js
- MongoDB
- Mongoose
- JWT Authentication
- bcrypt.js

## ⚙️ Installation

### Clone the repository

```bash
git clone https://github.com/your-username/BlogSphere.git
```

### Backend

```bash
cd backend
npm install
npm start
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```

---

## 🔑 Environment Variables

Create a `.env` file inside the backend folder.

```env
PORT=5000
MONGO_URI=your_mongodb_connection_string
JWT_SECRET=your_secret_key
```
